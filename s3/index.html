Managing Your S3 Bucket with AWS CLI: Listing and Deleting Objects

Amazon S3 is a powerful service that allows you to store and manage your data. In this guide, we will show you how to manage the objects stored in your S3 bucket using AWS CLI. Specifically, we'll walk you through the process of listing the contents of your bucket and deleting specific objects to keep your storage organized.

Prerequisites

Before you begin, make sure you have the following:

  • An AWS account with access to S3
  • A configured AWS CLI with the appropriate permissions to access your S3 bucket
  • Download and install the AWS CLI

Step 1: Install and Configure AWS CLI

First, you'll need to install the AWS CLI. You can follow the instructions in the official AWS documentation for your operating system:

  • For Windows: Install AWS CLI on Windows
  • For macOS: Install AWS CLI on macOS
  • For Linux: Install AWS CLI on Linux

Once installed, configure your AWS CLI with your credentials:

aws configure

You'll be prompted to enter the following information:

  • AWS Access Key ID: You can find this in your AWS account (IAM section).
  • AWS Secret Access Key: Also found in your IAM settings.
  • Default region name: e.g., us-east-1

Step 2: List the Contents of Your S3 Bucket

Once your AWS CLI is set up, use the following command to list all objects in your S3 bucket. Replace your-bucket-name with your actual bucket name:

aws s3 ls s3://your-bucket-name --recursive

Example output:


2025-01-07 12:30:02     410650 example-file1.jpg
2025-01-07 12:42:05     410650 example-file2.jpg
2025-01-07 12:45:57     410650 example-file3.jpg
2025-01-07 15:38:14      13089 example-file4.png
...

In this case, we have a bucket named your-bucket-name, and the objects are listed with their timestamps and sizes.

Step 3: Delete Specific Objects from Your S3 Bucket

Once you've identified the object you want to delete, use the s3 rm command to remove it from your bucket. For example, to delete example-file1.jpg, run:

aws s3 rm s3://your-bucket-name/example-file1.jpg

Example confirmation message:

delete: s3://your-bucket-name/example-file1.jpg

This confirms that the object has been successfully deleted from the bucket.

Conclusion

With these simple AWS CLI commands, you can effectively manage your S3 bucket by listing objects and deleting unwanted files. Regularly managing your S3 bucket can help you optimize storage and reduce unnecessary costs.
